# 🎓 Student Grade Tracker

## Overview

Student Grade Tracker is a Java Swing GUI application developed as part of the CodeAlpha Java Programming Internship.

The application allows users to add student names and grades, store records using ArrayList, and generate a summary report including average, highest, and lowest grades.

## Features

* Add student names and grades
* Store data using ArrayList
* Calculate average grade
* Find highest grade
* Find lowest grade
* Display summary report of all students
* Input validation and error handling
* User-friendly Java Swing GUI

## Technologies Used

* Java
* Swing
* ArrayList
* Event Handling

## Requirements Covered

✔ Input and manage student grades

✔ Calculate average score

✔ Calculate highest score

✔ Calculate lowest score

✔ Use ArrayList to store and manage data

✔ Display a summary report of all students
