
# 🎓 Student Grade Tracker

A Java Swing GUI application to manage and analyze student grades. Developed as part of the **CodeAlpha Java Programming Internship – Task 1**.

---

## 📋 Features

✅ Add student names and grades through an interactive GUI

✅ Store student records using **ArrayList**

✅ Calculate average grade automatically

✅ Find highest and lowest grades

✅ Generate a complete summary report

✅ Input validation and error handling

✅ User-friendly Java Swing interface

---

## 🖥️ Tech Stack

**Language:** Java

**GUI Framework:** Java Swing (`javax.swing`)

**Data Structure:** ArrayList

**IDE:** IntelliJ IDEA / Eclipse / VS Code

---

## 🚀 How to Run

### Prerequisites

* Java JDK 8 or higher
* Any Java IDE or Terminal

### Using Terminal

```bash
# Compile
javac task1.java

# Run
java task1
```

### Using an IDE

1. Open the project in IntelliJ IDEA, Eclipse, or VS Code.
2. Run `task1.java` as a Java Application.

---

## 🧭 How to Use

1. Enter the student's name in the **Student Name** field.
2. Enter the student's grade (0–100).
3. Click **Add Student** to save the record.
4. Repeat for multiple students.
5. Click **Show Report** to generate the complete report.
6. View:

   * Total Students
   * Average Grade
   * Highest Grade
   * Lowest Grade
